# Akeno Discord Theme

## Description
A custom Discord theme featuring a refined aesthetic based on the Glass Wave framework, with custom color grading and background integration.

## Installation
1. Ensure you are using a modded client such as Vencord or BetterDiscord.
2. Navigate to your theme settings.
3. Import the 'akeno.css' file or copy the raw code into your custom CSS editor.

## Recommendation: High Contrast Mode
For an optimal visual experience and improved text legibility, it is highly recommended to enable the High Contrast mode in Discord:
1. Open User Settings.
2. Go to the Accessibility tab.
3. Enable "High Contrast" mode.
