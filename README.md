# Akeno Discord Theme

🇫🇷 [Version Française](#fr) | 🇬🇧 [English Version](#en)

<a id="fr"></a>
## 🇫🇷 Description
Un thème Discord personnalisé offrant une esthétique raffinée basée sur le framework Glass Wave, avec une colorimétrie ajustée et une intégration d'arrière-plan personnalisée.

### Installation
1. Assurez-vous d'utiliser un client modifié tel que Vencord ou BetterDiscord.
2. Accédez à vos paramètres de thèmes.
3. Importez le fichier 'akeno.css' ou copiez le code source dans votre éditeur CSS personnalisé.

### Recommandation : Mode Haut Contraste
Pour une expérience visuelle optimale et une meilleure lisibilité, il est fortement recommandé d'activer le mode "Haut contraste" dans Discord :
1. Ouvrez les Paramètres utilisateur.
2. Allez dans l'onglet Accessibilité.
3. Activez le "Mode haut contraste".

### Crédits & Base
- Framework de base : [Glass-Wave](https://github.com/Elisniper/Glass-Wave)
- Image d'arrière-plan : [Akeno Wallpaper](https://wallpapercave.com/wp/wp7936990.jpg)
- Ajustements de couleur : Personnalisés

### Support
Si vous rencontrez des problèmes d'affichage ou des bugs, veuillez ouvrir une "Issue" sur ce dépôt.

## 🇬🇧 Description
A custom Discord theme featuring a refined aesthetic based on the Glass Wave framework, with custom color grading and background integration.

### Installation
1. Ensure you are using a modded client such as Vencord or BetterDiscord.
2. Navigate to your theme settings.
3. Import the 'akeno.css' file or copy the raw code into your custom CSS editor.

### Recommendation: High Contrast Mode
For an optimal visual experience and improved text legibility, it is highly recommended to enable the High Contrast mode in Discord:
1. Open User Settings.
2. Go to the Accessibility tab.
3. Enable "High Contrast" mode.

### Credits & Base
- Base framework: [Glass-Wave](https://github.com/Elisniper/Glass-Wave)
- Background image: [Akeno Wallpaper](https://wallpapercave.com/wp/wp7936990.jpg)
- Color adjustments: Custom

### Support
If you encounter any display issues or bugs, please open an issue in this repository.
